# PixelCraft 🎨

PixelCraft is a feature-rich **Pixel Art & Animation Creation Tool** built using HTML5 Canvas. As a **Progressive Web App (PWA)**, it supports offline access, is mobile-friendly, and offers an intuitive user experience for artists and animators alike.

## Features

- **Easy-to-use Tools**: A variety of drawing, shape, and editing tools for precise control over pixel art creation.
- **GIF Animation**: Simple tools for frame-by-frame animation and GIF creation.
- **Offline Compatible**: As a PWA, PixelCraft works offline and can be added to your home screen.
- **Advanced Customization**: Import images, set color transparency, and convert images to pixel art.
  
---

## Tools Overview 🛠️

### Drawing & Editing Tools

- **Pencil**: Freehand drawing of pixels.
  ![Pencil Tool](https://user-images.githubusercontent.com/17960677/96023223-b83b8800-0e6f-11eb-8af1-0cfab42c19d7.png)

- **Eraser**: Erase individual pixels with precision (fixed at 1x1 dimensions).
  ![Eraser Tool](https://user-images.githubusercontent.com/17960677/96023372-f5077f00-0e6f-11eb-965f-815b37f0d7b2.png)

- **Paint Tool**: Flood fills an area with a new color (recommended for dimensions under 128x128).
  ![Paint Tool](https://user-images.githubusercontent.com/17960677/96023668-66473200-0e70-11eb-81a2-9313b1580926.png)

- **Line Tool**: Draws straight lines between two points using Bresenham's Line Algorithm.
  ![Line Tool](https://user-images.githubusercontent.com/17960677/96023893-b625f900-0e70-11eb-9b16-ac0d68f9423c.png)

- **Circle Tool**: Creates circles with specified center and radius (using the Midpoint Circle Algorithm).
  ![Circle Tool](https://user-images.githubusercontent.com/17960677/96024358-3d736c80-0e71-11eb-8c1a-acb528f027b6.png)

- **Ellipse Tool**: Draws ellipses with given center and specified x and y radii.
  ![Ellipse Tool](https://user-images.githubusercontent.com/17960677/96024638-a0fd9a00-0e71-11eb-9171-16944bf95b40.png)

### Animation Tools 🎞️

- **Add Frame**: Saves the current canvas as a frame for animation, with a default delay of 100 ms per frame.
  ![Frames](https://user-images.githubusercontent.com/17960677/96025150-5f212380-0e72-11eb-9dce-5335a1cddcb7.png)

- **View Frame**: Access and manage frames in a stack. Load or delete frames as needed.
  ![Frame Panel](https://user-images.githubusercontent.com/17960677/96025802-48c79780-0e73-11eb-8ca1-4948226804ad.png)

### Utility Tools

- **Undo/Redo**: Basic functionality to undo or redo one pixel at a time—ideal for small corrections.
  ![Undo/Redo](https://user-images.githubusercontent.com/17960677/96026461-23875900-0e74-11eb-8682-479b33894a4f.png)

- **Clear Window**: Clears the canvas completely.
  ![Clear Window](https://user-images.githubusercontent.com/17960677/96026664-73662000-0e74-11eb-8111-c46f54375056.png)

---

## Advanced Features 🔧

- **Import Image**: Converts any uploaded image into pixel art with specified dimensions.
  ![Mona Lisa](https://user-images.githubusercontent.com/17960677/96027394-71509100-0e75-11eb-8260-c1ed426805bc.png)

- **Save Image**: Download your creation as a `.png` file with custom scaling.
  
- **Color Palette & Transparency**: Select colors from a built-in palette, and set color transparency for added effects. 
  ![Color Palette](https://user-images.githubusercontent.com/17960677/96028508-fdaf8380-0e76-11eb-8502-18d49d96d059.png)
